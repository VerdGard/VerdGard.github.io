ManTa 是一款基于 LuaAppx Pro 框架的动漫数据采集 APP，由 SoloSu 开发。 通过 LuaJava 桥接、XPath 精准匹配与 Jsoup HTML 解析引擎， 实现对多源动漫数据的高效聚合与搜索。

Core Engine

技术核心

每一行代码都为动漫数据而生。从底层解析到上层展示，ManTa 构建了一套完整的数据管线。

稳定

XPath 精准匹配

利用稳定的 XPath 语法定位目标数据节点，支持复杂层级结构与属性筛选，确保每次采集结果的精确性和一致性。

高效

Jsoup 解析引擎

基于 Jsoup 实现 HTML 文档的快速解析与 DOM 遍历，支持 CSS Selector 与 XPath 双模式，大幅提升数据提取效率。

桥接

LuaJava 桥接

通过 LuaJava 框架实现 Lua 与 Java 的双向调用，让业务逻辑层与数据解析层无缝协作，同时保留 Lua 的灵活性与 Java 生态的成熟库支持。

框架

LuaAppx Pro 框架

基于 LuaAppx Pro 运行时，提供完整的 Android 原生能力封装。支持多线程调度、本地缓存管理、网络请求池与插件化扩展体系。框架层面保障了数据采集任务的稳定性与低功耗运行。

开箱即用

官方配置驱动

内置经过验证的官方采集配置，涵盖主流动漫站点。安装即用，无需手动编写规则。预置配置持续更新，适配站点结构变化。

开发中

自定义规则（即将推出）

规划中的开放规则编辑器将允许用户通过 UI 配置自定义采集规则，包括 URL 模板、XPath 表达式与数据映射，实现真正的个性化采集。

Pipeline

工作原理

从 URL 输入到结构化数据输出，ManTa 的数据管线分为四个清晰的阶段。

配置匹配

用户输入动漫名称或 URL 后，系统自动匹配预置的官方配置模板，识别目标站点类型与数据结构特征。匹配过程支持模糊权重算法，容忍 URL 格式差异。

luajava → config matcher

HTML 获取与解析

使用 LuaAppx Pro 网络请求池异步获取目标页面 HTML，交给 Jsoup 引擎解析为 DOM 树。支持请求重试、超时控制与 Cookie 持久化。

Jsoup.parse(html)

XPath 数据抽取

根据配置中的 XPath 表达式精准定位目标数据节点，提取标题、封面、评分、标签、章节列表等字段。支持相对路径、属性筛选与文本归一化。

xpath("//div[@class='detail']/h1/text()")

结构化输出

抽取的数据经过清洗、去重与格式标准化后，聚合为统一的数据模型，供 UI 层渲染展示。整个管线通过 LuaJava 桥接协作，纯异步、低延迟。

callback → UI render
