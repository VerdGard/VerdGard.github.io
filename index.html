三步完成数据采集

1

初始化项目 — 运行 mania init 创建爬虫骨架

2

配置规则 — 用 YAML 或可视化编辑器定义目标页面和数据字段

3

运行并导出 — 执行 mania run 获取结构化数据

架构

数据流 清晰可见

从配置到输出，每一步都透明可控。

01

配置规则

通过 YAML 文件或可视化编辑器定义目标 URL、选择器和数据字段映射规则。

02

启动引擎

LuaAppx Pro 运行时加载规则，初始化抓取队列和解析管道。

03

处理数据

多线程并发抓取，智能解析引擎自动提取结构化数据，支持管道转换。

04

输出结果

数据可导出为 JSON / CSV / SQLite，或直接打包为独立 Android APP。
